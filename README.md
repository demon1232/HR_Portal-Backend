# HR Portal (Django + Android App)

A complete HR management system with web dashboard and Android application.

---

## Live Demo

https://nabeel789.pythonanywhere.com

---

## Features

* Employee Login System
* Attendance Management
* RFID Attendance Integration
* Salary and Payslip System
* Leave Management
* HR Dashboard
* Employee Records Management

---

## Android App

This project also includes an Android application built using WebView.

* Loads live HR portal
* Simple and fast interface
* Compatible with most Android devices

---

## Tech Stack

### Backend

* Python
* Django
* SQLite

### Frontend

* HTML
* CSS
* JavaScript

### Mobile App

* Android (Kotlin)
* WebView

---

## Installation (Local Setup)

```bash
git clone https://github.com/demon1232/HR_Portal-Backend.git
cd HR-Portal-Backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

---

## Admin Access

```bash
python manage.py createsuperuser
```
